Material variant strategyKeep high-quality keyword combinationsStrictly trim shader variantsReduce package size and runtime switching cost
Texture and sampling strategyCore assets can use higher resolution and sampling levelsUnified ASTC tiers + capped high-order samplingStabilize memory footprint and bandwidth
Quality tier strategyFine-grained High / Ultra tiersLow / Mid prioritized, key features enabled via whitelistBuild a configurable cross-platform quality matrix

3.3 Comparison Conclusions

  1. The current delivery focus includes 27 items, with priority on getting the Mobile pipeline fully production-ready (motion vectors, temporal upscaling, and TerrainToMesh, among others).
  2. The next-phase enhancement scope also includes 27 items, covering common realistic material families inspired by UE4 projects and PC-side capability backfill.
  3. The future exploratory track includes 4 items, focused on feasibility validation for additional light-source shadows, decals/DBuffer, and multiple reflection captures.
  4. There are 7 baseline items, and all of them are already stable capabilities in the current rendering pipeline.
